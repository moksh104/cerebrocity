# CerebroCity 🏙️

**CerebroCity – Build Your Focus Empire**

CerebroCity is a gamified productivity app that helps users stay focused while building a virtual city. Each focus session earns XP and unlocks new buildings, turning productivity into a fun and rewarding experience.

---

## 🚀 Features

- ⏱ Focus timer for productive sessions
- ⭐ XP and leveling system
- 🏙 City-building progression
- 🔓 Unlock buildings as you focus
- 📊 Track productivity progress
- 🎮 Game-like user experience

---

## 🏗 How It Works

1. Start a focus session.
2. Complete the session to earn XP.
3. Level up and unlock new buildings.
4. Watch your city grow as you stay productive.
